How it works:

This application uses YOLOv11 (the latest version) to detect objects in real-time from your webcam. The model identifies common objects and displays bounding boxes around them with labels.

Features: Enhanced accuracy and speed with YOLOv11 technology, focused on detecting food and dining-related items.

Note: The first run might take a moment as the YOLOv11 model is loaded.
